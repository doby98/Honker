## :pushpin: Honker
>혼자서도 즐길 수 있는 포커 게임
</br>

## 1. 제작 기간 & 참여 인원
- (2022. 3. 21.~2022. 4. 6.)
- 개인 프로젝트
</br>

## 2. 사용 기술
- JAVA 8
</br>

## 3. 핵심 기능
>프로젝트의 핵심 기능은 카드패에 따른 족보 출력 및 승패 판정입니다.
>
>사용자는 기본적인 포커 룰에 따라 각 3차례씩 카드를 받고, 베팅을 진행할 수 있습니다.
>
>게임의 마지막까지 베팅을 완료한 경우에는 완성된 내 패와 비공개된 상대방의 패를 비교하여,
>
>이용자가 베팅한 금액의 두배를 얻거나, 잃을 수 있습니다.
</br>
